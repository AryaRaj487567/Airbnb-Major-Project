# 🏡 WanderLust – Airbnb Clone

A full-stack Airbnb-inspired web application where users can discover, create, edit, review, and manage vacation rental listings. Built using the MERN stack principles with Node.js, Express.js, MongoDB Atlas, EJS, Bootstrap, Cloudinary, Passport.js authentication, and Leaflet maps.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AryaRaj487567/Airbnb-Major-Project.git
```

Go into project directory

```bash
cd Airbnb-Major-Project
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
ATLASDB_URL=

SECRET=

CLOUD_NAME=

CLOUD_API_KEY=

CLOUD_API_SECRET=
```

Run the application

```bash
node app.js
```

Visit

```
http://localhost:8080
```
